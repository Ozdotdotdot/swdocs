## Quick Start (5 Steps)

1. Navigate to the [Dashboard page](http://localhost:3000/dashboard)
2. Select a **View Type** (State or Tournament)
3. Enter a **State** code (e.g., `GA`, `NY`, `CA`)
4. Choose a **Timeframe** (30 days, 60 days, or 3 months)
5. Click **Apply Filters**

<!-- SCREENSHOT: Dashboard with empty state showing the filter panel and "No data yet" message -->

The scatter chart will populate with player data from your selected region.

## Your First Visualization

### Example: New York State Players

1. Open the dashboard
2. Keep **State** view selected
3. Enter `NY` in the State field
4. Select **Last 3 months** from the timeframe dropdown
5. Click **Apply Filters**

You should see a scatter plot with New York players, showing their win rates versus opponent strength.

<!-- SCREENSHOT: Completed NY state visualization with data populated -->

## What You'll See

After applying filters, you'll get:

- **Scatter chart** - Each dot represents a player
- **Data table** - Detailed list below the chart with player stats
- **Row count** - Number of players displayed
- **Interactive tooltips** - Hover over dots to see player details
